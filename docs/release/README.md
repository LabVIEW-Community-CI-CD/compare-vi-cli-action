# Release Operations Docs

This directory holds the active release helper documents that operators and future agents should consult during release
work.

## Contents

- `PR_NOTES.md` - release PR summary helper
- `TAG_PREP_CHECKLIST.md` - tag preparation checklist
- `POST_RELEASE_FOLLOWUPS.md` - post-release backlog tracker
- `ROLLBACK_PLAN.md` - rollback procedure reference

Historical release notes, archived PR descriptions, and version-specific release checklists live under
`../archive/releases/`.
